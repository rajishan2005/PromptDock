PromptDock — Privacy Policy

Last updated: July 2026

PromptDock is a browser extension that adds a PDF viewing, cropping, and question-extraction panel to ChatGPT (chatgpt.com / chat.openai.com). This page explains what data the extension touches and what it does with it.

Summary

PromptDock does not collect, transmit, or sell any personal data. Everything the extension does happens locally in your browser.

What data PromptDock handles, and where it stays

  • PDF files you upload are read and rendered entirely in your browser to show the preview, let you crop, and extract question text. If you leave the "save to library" behavior on, the file is stored using chrome.storage.local — a storage area that lives only on your own device. It is never uploaded to any server operated by PromptDock or its developer.
  • Cropped question images are generated locally and inserted directly into the ChatGPT chat box you're already using — they pass through the same channel any file you'd manually paste into ChatGPT would, and are not routed through any separate PromptDock server (there isn't one).
  • Saved chat links (the "🔗 Saved Chats" feature) store only the URL of a ChatGPT conversation tab you explicitly chose to save, plus the name of the PDF you were working from. This is stored locally the same way, and is never sent anywhere else.
  • Autosolve automates typing and submitting text into ChatGPT's own input box — it does not read or store ChatGPT's responses, and does not communicate with any third-party service.

What PromptDock does NOT do

  • It does not have its own backend server — there is nothing to send your data to, even if it wanted to.
  • It does not use analytics, tracking pixels, or third-party advertising SDKs.
  • It does not sell or share data with advertisers or data brokers.
  • It does not read or interact with any site other than chatgpt.com and chat.openai.com.

Permissions explained

  • activeTab / scripting — used to inject the PromptDock panel into the ChatGPT page you're currently viewing.
  • storage / unlimitedStorage — used to save PDFs you upload and chat links you choose to save, locally on your device, so you don't have to re-upload files each time. PDFs can be several megabytes each, which is why the higher storage allowance is requested.
  • Host permission for chatgpt.com / chat.openai.com — required so the extension can display its panel and insert content into ChatGPT's chat box on those specific sites only.

Removing your data

You can delete any saved PDF or chat link directly inside the extension's panel (📚 Library and 🔗 Saved Chats sections each have a delete button). Uninstalling the extension removes all locally stored data along with it.
